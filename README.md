# stringify-keys [![NPM version](https://badge.fury.io/js/stringify-keys.svg)](http://badge.fury.io/js/stringify-keys)

> Build an array of key paths from an object.

## Install
## Install with [npm](npmjs.org)

```bash
npm i stringify-keys --save
```

## Run tests

```bash

```

